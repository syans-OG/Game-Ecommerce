# GameVault: Premium Digital Game Marketplace

GameVault is a premium digital game e-commerce platform designed to provide gamers with a seamless discovery, purchase, and management experience. From indie gems to triple-A blockbusters, GameVault is the ultimate hub for modern gaming.

## ✨ Features

- **🎯 Discovery Excellence**: Curated collections, personalized recommendations, and a powerful search system.
- **🛒 Seamless Checkout**: Secure and fast payment processing with multiple options.
- **📚 Unified Library**: A centralized hub to access and manage all your purchased titles.
- **❤️ Wishlist & Alerts**: Save games for later and get notified when they go on sale.
- **👤 User Management**: Full profile customization, order history, and secure authentication.
- **🌟 Community Reviews**: Genuine player feedback to help you make informed decisions.

## 🚀 Pages Included

- **🏠 Home Page**: The gateway to the latest and greatest in gaming.
- **🔍 Game Catalog**: Advanced filtering and search for deep discovery.
- **📄 Game Detail**: Comprehensive info, system requirements, and galleries.
- **➕ Cart & Checkout**: Optimized conversion funnel from cart to completion.
- **📁 Player Library**: Your digital collection, ready for download.
- **⚙️ Profile & Orders**: Manage your account and track your purchases.
- **🔐 Auth**: Secure Login and Registration flows.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Modern, responsive styling with premium aesthetics.
- **JavaScript**: Dynamic interactions and client-side logic.

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/syans-OG/Game-Ecommerce.git
   ```
2. Open `html/index.html` in your browser to start exploring.
